#  App de Meditación y Mindfulness

##  Dominio
Salud y Bienestar

##  Objetivo del Proyecto
Modelar una aplicación de meditación y mindfulness utilizando TypeScript,
aplicando interfaces, types, type unions, type literals y funciones tipadas,
siguiendo las instrucciones del proyecto semanal.

##  Entidades Modeladas
- **MeditationSession**: representa una sesión de meditación.
- Estados de sesión: `planned`, `completed`.
- Tipos de meditación: `guided`, `silent`, `breathing`.

##  Funcionalidades Implementadas
- Crear sesiones de meditación.
- Listar todas las sesiones creadas.
- Filtrar sesiones según su estado.
